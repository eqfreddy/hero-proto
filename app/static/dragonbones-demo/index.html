What you're looking at

Mecha 1004B is the official DragonBones flagship sample — a humanoid mech rig with weapon swap and a full action vocabulary. Same skeletal-animation tech as Spine, but free.

The action set already maps cleanly onto our combat resolver:

  • idle → between turns / waiting
  • walk → entrance, retreat
  • attack_01 → BASIC damage event
  • skill_01 / skill_03 / skill_04 / skill_05 → SPECIAL types (DAMAGE / AOE_DAMAGE / HEAL / BUFF)
  • hit → DAMAGE incoming
  • death → DEATH event
  • victory → end-of-battle WIN
Pros (over Plan A stick figures)
  • Smooth bone interpolation, mesh deform supported
  • Bone events fire at exact frames — perfect for syncing damage popups + particle hits
  • ~8KB JSON + 1MB atlas per rig; reuse the rig across all heroes via skin slots
  • Free editor + free runtime, MIT/Apache licensing
  • Phaser plugin (raksa/phaser-dragonbones) still active as of Dec 2025
Cons / open questions
  • Editor learning curve (~2 days for rigging from scratch)
  • Phaser plugin is community-maintained, not first-party
  • Three rigs ≠ thirty bespoke heroes — every ATK still moves identically; head/weapon swap is your differentiator
  • Pixi 4.6 here for the demo; production would migrate to Pixi 8.x or run alongside Phaser via the plugin
